Ana Ochonga (33 años)

| Programadora BI | | Estudiante de Desarrollo Web |

Creo que este trabajo nos permite tomar conciencia de cuán importante es el libre acceso a la Web para todos y que parece demasiado común al momento de desarrollar una Página Web no tener en cuenta diferentes situaciones (No videntes, Visión Reducida, Problemas motrices, etc.) las cuales no son pensadas, ya que se desarrolla para un usuario "Tipo". Con el desarrolló del IEFI entendí que es importante contemplar el análisis de la accesibilidad web como premisa al momento de crear o aplicar ciencias de la información. Una navegación intuitiva debería ser otro de los objetivos ya que, en algunos casos, al usarse tecnicismos o estructuras muy complejas generan que aquellas personas que no tengan un uso continuo de la web queden excluida.
